# `Method.toClient`

Extends a method with client-side credential creation logic.

## Usage

```ts twoslash
import { Credential, Intent, Method, MethodIntent, z } from 'mpay'

const charge = MethodIntent.fromIntent(Intent.charge, {
  method: 'stripe',
  schema: {
    credential: {
      payload: z.object({ signature: z.string() }),
    },
  },
})

const stripe = Method.from({
  name: 'stripe',
  intents: { charge },
})

declare function sign(challenge: unknown): Promise<string>
// ---cut---
const client = Method.toClient(stripe, {
  async createCredential({ challenge }) {
    const signature = await sign(challenge)
    return Credential.serialize({ 
      challenge, 
      payload: { signature } 
    })
  },
})
```

### With Context

Pass additional context (like credentials) to the `createCredential` function.

```ts twoslash
import { Credential, Intent, Method, MethodIntent, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
    }),
  },
})

const stripeCharge = MethodIntent.fromIntent(charge, {
  method: 'stripe',
  schema: {
    credential: {
      payload: z.object({ signature: z.string() }),
    },
  },
})

const stripe = Method.from({
  name: 'stripe',
  intents: { charge: stripeCharge },
})

declare function signWith(key: string, challenge: unknown): Promise<string>
// ---cut---
const client = Method.toClient(stripe, {
  context: z.object({
    privateKey: z.string(),
  }),
  async createCredential({ challenge, context }) {
    const signature = await signWith(context.privateKey, challenge)
    return Credential.serialize({ 
      challenge, 
      payload: { signature } 
    })
  },
})
```

## Return Type

```ts
type ReturnType = Client<name, intents, context>
```

A client-side payment method with credential creation logic.

## Parameters

### method

- **Type:** `Method`

The base method to extend.

### options

#### context (optional)

- **Type:** `z.ZodMiniType`

Schema for per-request context passed to `createCredential`.

#### createCredential

- **Type:** `(parameters: { challenge, context? }) => Promise<string>`

Function that creates a serialized credential from a challenge.
