# `Intent.subscription`

Subscription intent for recurring periodic payments.

## Usage

```ts twoslash
import { Intent } from 'mpay'

// Use the built-in subscription intent
const subscription = Intent.subscription
```

## Type

```ts
const subscription: Intent
```

## Schema

### request

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `amount` | `string` | ✓ | Amount per period |
| `currency` | `string` | ✓ | Currency identifier |
| `period` | `string` | ✓ | Payment period (e.g., "P1M" for monthly) |
| `cycles` | `number` | | Number of payment cycles (positive integer) |
| `description` | `string` | | Human-readable description |
| `expires` | `string` | | Subscription end date (ISO 8601) |
| `externalId` | `string` | | External reference ID |
| `recipient` | `string` | | Payment recipient |

## Reference

See the [subscription intent specification](https://github.com/tempoxyz/payment-auth-spec/blob/main/specs/intents/draft-payment-intent-subscription-00.md).
