# `Method.from`

Creates a payment method from parameters.

## Usage

```ts twoslash
import { Method, MethodIntent, Intent, z } from 'mpay'

const charge = MethodIntent.fromIntent(Intent.charge, {
  method: 'custom',
  schema: {
    credential: {
      payload: z.object({ signature: z.string() }),
    },
  },
})

const method = Method.from({
  name: 'custom',
  intents: { charge },
})
```

## Return Type

```ts
type ReturnType = Method<name, intents>
```

A payment method definition.

## Parameters

### parameters

#### name

- **Type:** `string`

Payment method name (e.g., "tempo", "stripe").

#### intents

- **Type:** `Record<string, MethodIntent>`

Map of intent names to method intents.
