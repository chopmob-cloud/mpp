# `Method.toServer`

Extends a method with server-side verification logic.

## Usage

```ts twoslash
import { Intent, Method, MethodIntent, Receipt, z } from 'mpay'

const charge = MethodIntent.fromIntent(Intent.charge, {
  method: 'stripe',
  schema: {
    credential: {
      payload: z.object({ signature: z.string() }),
    },
  },
})

const stripe = Method.from({
  name: 'stripe',
  intents: { charge },
})

declare function verifySignature(credential: unknown): Promise<boolean>
// ---cut---
const server = Method.toServer(stripe, {
  async verify({ credential }) {
    const isValid = await verifySignature(credential)
    
    return Receipt.from({
      method: 'stripe',
      status: isValid ? 'success' : 'failed',
      timestamp: new Date().toISOString(),
      reference: '0x...',
    })
  },
})
```

### With Context

Pass additional context (like configuration) to the `verify` function.

```ts twoslash
import { Intent, Method, MethodIntent, Receipt, z } from 'mpay'

const charge = Intent.from({
  name: 'charge',
  schema: {
    request: z.object({
      amount: z.string(),
      currency: z.string(),
    }),
  },
})

const stripeCharge = MethodIntent.fromIntent(charge, {
  method: 'stripe',
  schema: {
    credential: {
      payload: z.object({ signature: z.string() }),
    },
  },
})

const stripe = Method.from({
  name: 'stripe',
  intents: { charge: stripeCharge },
})

declare function verifyOnChain(rpcUrl: string, credential: unknown): Promise<boolean>
// ---cut---
const server = Method.toServer(stripe, {
  context: z.object({
    rpcUrl: z.string(),
  }),
  async verify({ credential, context }) {
    const isValid = await verifyOnChain(context.rpcUrl, credential)
    
    return Receipt.from({
      method: 'stripe',
      status: isValid ? 'success' : 'failed',
      timestamp: new Date().toISOString(),
      reference: '0x...',
    })
  },
})
```

## Return Type

```ts
type ReturnType = Server<name, intents, context>
```

A server-side payment method with verification logic.

## Parameters

### method

- **Type:** `Method`

The base method to extend.

### options

#### context (optional)

- **Type:** `z.ZodMiniType`

Schema for per-request context passed to `verify`.

#### request (optional)

- **Type:** `(parameters) => request`

Transform request before challenge creation.

#### verify

- **Type:** `(parameters: { credential, context, request }) => Promise<Receipt>`

Function that validates a credential and returns a receipt.
